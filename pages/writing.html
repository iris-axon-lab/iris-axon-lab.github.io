Quality Architecture for Agentic Skills and Tools
End-to-end quality in agentic tool use is a multiplicative function of three layers: contract quality, execution quality, and orchestration-time selection. Optimizing selection alone yields limited gains if the upstream layers remain unaddressed.

A Four-Tier Memory Taxonomy for Enterprise Agentic Systems
Four memory tiers — semantic, episodic, procedural, and prospective — mapped to enterprise agentic use cases. The session boundary stress test: if commitments don't survive a session end without the user restating them, you have a prospective memory gap.
